Google BigQuery 활용을 위한 종합 가이드 및 실습 예제 모음집입니다.

한국어로 작성된 상세한 문서와 실제 업무에서 활용할 수 있는 예제 코드를 제공합니다.

> **면책 조항**: 이 워크북의 내용은 학습 및 참고 목적으로 제공되며, 정확성이나 완전성을 보장하지 않습니다.

## 주요 특징

- **한국어 문서**: 모든 문서가 한국어로 작성
- **실무 중심**: 실제 업무에서 바로 활용할 수 있는 예제
- **포괄적 커버리지**: BigQuery의 거의 모든 기능을 다룸
- **코드 예제**: SQL, Shell Script, Python, YAML 등 다양한 형태

## 시작하기

```bash
# Google Cloud CLI 인증
gcloud auth login
gcloud config set project YOUR_PROJECT_ID

# BigQuery CLI 테스트
bq ls
```
